**Infaplastic** es una plataforma web interactiva y moderna diseñada para la comercialización y distribución de empaques desechables sostenibles y ecológicos para restaurantes, comercios y negocios en Venezuela.

---

## 🚀 Características Principales

- **Catálogo Interactivo con Filtros**: Navegación por categorías (Contenedores, Vasos, Cubiertos, Eco-pack) con filtrado instantáneo en tiempo real.
- **Alternador de Precios (Detal / Mayorista)**: Permite a los clientes alternar entre modalidad de compra minorista y compras por volumen con precios especiales B2B.
- **Calculadora de Ahorro B2B**: Herramienta interactiva con deslizador (*range slider*) que calcula automáticamente el ahorro estimado mensual y anual al adquirir suministros al mayor.
- **Carrito de Compras Flotante (Cart Drawer)**: Experiencia de compra interactiva sin recarga de página, con barra de progreso dinámica para alcanzar el envío gratuito.
- **Modales de Vista Rápida (Quickview)**: Inspección detallada de productos, precios por volumen y adición directa al carrito.
- **Carrusel Promocional & Banner de Cuenta Regresiva**: Destacados dinámicos con ofertas limitadas.
- **Diseño Responsive & UI de Alto Nivel**: Construido con tipografías avanzadas (*Archivo* y *Inter*), variables de color CSS tailor-made, animaciones fluidas y modo adaptativo a dispositivos móviles.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5 Semántico**: Estructuración accesible y optimizada para SEO.
- **CSS3 Vanilla**: Sistema de diseño basado en variables CSS (`:root`), Flexbox, CSS Grid, Glassmorphic backdrop filters y micro-animaciones en CSS puro.
- **JavaScript (ES6+)**: Gestión dinámica de estado en el cliente (carrito, modales, calculadora de ahorro, filtros de catálogo y sliders).
- **Google Fonts**: Tipografías de alto impacto (`Archivo` y `Inter`).

## ⚙️ Cómo Visualizar o Ejecutar Localmente

No se requieren dependencias externas ni compiladores para ejecutar la aplicación.

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/jotarampini-cell/Infaplastic.git
   cd Infaplastic
   ```

2. **Abrir en el navegador**:
   - Haz doble clic sobre el archivo `index.html`.
   - O mediante una extensión como **Live Server** en VS Code o ejecutando un servidor local básico:
     ```bash
     npx serve .
     ```

## 🌐 Despliegue en GitHub Pages

Para publicar este proyecto gratuitamente con GitHub Pages:

1. Ve a la pestaña **Settings** de tu repositorio en GitHub (`jotarampini-cell/Infaplastic`).
2. En la barra lateral izquierda, selecciona **Pages**.
3. En **Source**, selecciona `Deploy from a branch` y elige la rama `main` (directorio `/root`).
4. Haz clic en **Save**. En pocos momentos, la página estará accesible públicamente.
